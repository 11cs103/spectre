flexbox layout

layout.less includes flexbox based grid system with 12 columns.

Add the class columns to a container. Add any element you want with class column inside the container. These columns will take up the space equally. You can specific the width of each column by adding class col-[1-12].

tables

tables.less sets default styles for tables and data sets.

Add the class <table> to any table element. The class will add padding, border and emphasized table header. Adding the class selected to a <tr> element will highlight the whole table row.

buttons

buttons.less includes simple button style for actions in different types and sizes.

Add the class btn to <a>, <input> or <button> elements for a default button. There are classes btn-primary and btn-link for predefined primary and link buttons. A button with the class loading can show loading indicator.

button sizes

Add the class btn-sm or btn-lg for small or large size. Icons with the class icon can be correctly rendered in each button size. Also, you can add the class btn-block for a full-width button.

button groups

If you want to use buttons as a group, add the class btn-group to the container.

tooltips

tooltips.less is a label-like component that appears on hover and focus.

Add the class tooltip and the attribute data-tooltip, which contains the tooltip content, to non self closing elements. And add the class name tooltip-right, tooltip-bottom or tooltip-left to define the position of a tooltip. By default, the tooltip appears above the element.

<button class="btn btn-primary tooltip tooltip-right" data-tooltip="Lorem ipsum dolor sit amet">right tooltip</button>

toasts

toasts.less is used to show alert or information to users.

Add a container element with the class toast. You can add any text within the container, and even icons. You may also add a close button with the class btn-clear if you need.

badges

badges.less is often used as unread number indicator.

Add the class badge to non self closing elements. And add the attribute data-badge to define the content of a badge. The badge will appear in the top-right direction of the element.
